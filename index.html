    App trivia

    image
    Description
    Es juego de trivia que tiene dos modos de juego:
    El primer modo es el modo default, que consiste en contestar las preguntas y conseguir puntos. Consta de tres niveles de dificultad, a medida que juegas y aciertas las preguntas subes de nivel y así mismo las preguntas acertadas dan mas puntos.
    El segundo modo de juego te permite filtrar las preguntas pero no optienes puntos ni niveles.
    Characteristics
    • Api consuming, async functions
    • Original and dynamic music depending on level
    • Leveling system in which your level increase or decrease depending on your right and wrong answers.
    Technologies

    HTML5, CSS y JavaScript Vanilla
